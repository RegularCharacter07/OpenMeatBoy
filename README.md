
# OpenMeatBoy

OpenMeatBoy is an Open source reimplementation of Super Meat Boy.

This project is just getting started. I'm making this repository public so people can see my progress.

I'm warning you now that this project may never be finished. I'm doing it for fun, and I'm not an expert programmer.


# Building

## Windows (The only one for now)

- Clone the code.
- Install Visual Studio 2026.
- You need to have vcpkg.
- Open the OpenMeatBoy folder in Visual Studio.
